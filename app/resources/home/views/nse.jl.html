Not Software Engineering

Our web ring consists of Not Software Engineers. This is a little inside joke at our study, because we all study Network & Systems Engineering - opposing to Software Engineering - which handles, well, networks and systems, which is more low level. Of course we still write a lot of software (our websites being the prime example), but it's nice to make fun of people, especially because there are so many more software engineers than system engineers.

Ye old web

All of our websites are styled in web 1.0.
